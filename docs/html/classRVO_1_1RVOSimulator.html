Parameters
[in]timeStepThe time step of the simulation. Must be positive.
[in]neighborDistThe default maximum distance center-point to center-point to other agents a new agent takes into account in the navigation. The larger this number, the longer he running time of the simulation. If the number is too low, the simulation will not be safe. Must be non-negative.
[in]maxNeighborsThe default maximum number of other agents a new agent takes into account in the navigation. The larger this number, the longer the running time of the simulation. If the number is too low, the simulation will not be safe.
[in]timeHorizonThe default minimal amount of time for which a new agent's velocities that are computed by the simulation are safe with respect to other agents. The larger this number, the sooner an agent will respond to the presence of other agents, but the less freedom the agent has in choosing its velocities. Must be positive.
[in]timeHorizonObstThe default minimal amount of time for which a new agent's velocities that are computed by the simulation are safe with respect to obstacles. The larger this number, the sooner an agent will respond to the presence of obstacles, but the less freedom the agent has in choosing its velocities. Must be positive.
[in]radiusThe default radius of a new agent. Must be non-negative.
[in]maxSpeedThe default maximum speed of a new agent. Must be non-negative.
